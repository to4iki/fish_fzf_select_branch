# fish_fzf_select_branch

Select git branch checkout

## Install

With [fisher]
With [fzf]

```
fisher install to4iki/fish_fzf_select_branch
```

## Usage
Keybindings | Remarks
--- | ---
Ctrl-b | Select branch & checkout

[fisher]: https://github.com/jorgebucaran/fisher
[fzf]: https://github.com/junegunn/fzf
